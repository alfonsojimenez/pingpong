# Pingpong

Classic pingpong game written in Ruby. It is possible to play it in single
player or multiplayer mode.

![Pingpong](http://i.imgur.com/Su0AZmW.png)

## Setup

```
git clone git@github.com:alfonsojimenez/pingpong.git
bundle install
./bin/pingpong
```

Enjoy! :)
